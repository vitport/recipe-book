# Recipe Book Project — Zeev & Vitaly

## Stack
- Frontend: HTML/CSS/JavaScript (single file)
- Backend: Node.js + Express (server.js)
- Database: recipes.json (flat file)
- AI: Ollama + Mistral 7B (local)
- Testing: test.html (66+ AI-powered tests)

## Current Features
- Multilingual recipe book (Hebrew/English/Russian)
- RTL support for Hebrew
- AI parsing with Mistral (paste text, URL, PDF, photo)
- Image picker with Mistral food detection and ranking
- Photo manager popup (add/delete/reorder photos)
- Node.js Express server with auto-save
- Network sharing (http://192.168.1.2:8080)
- Sorting, filtering, search
- PDF import and export
- 66+ AI-powered tests with 100% pass rate
- Favicon, logging, export reports
